  1. Profile

    정보와 사람 사이를 연결하는 학생

    광운대학교 정보융합학부에서 데이터, 소프트웨어, 사용자 경험을 함께 배우고 있습니다. 기능을 만드는 데서 멈추지 않고, 그 기능이 사람에게 어떤 흐름으로 전달되는지까지 설계하는 개발자로 성장하고 있습니다.

    • Software
    • Data
    • AI
    • UI/UX

  2. 2021

    정보융합학부에서 여정을 시작하다

    데이터, 개발, 인공지능처럼 서로 다른 분야가 하나의 문제를 해결하기 위해 연결되는 방식을 처음 배웠습니다. 이 시기는 기술을 따로 익히기보다, 지식 사이의 관계를 보는 시야를 넓힌 출발점이었습니다.

    • Information Convergence
    • Problem Framing

  3. 2022

    코드와 데이터의 언어를 익히다

    Java와 Python을 통해 프로그램의 구조를 이해하고, 데이터 처리와 분석의 기초를 다졌습니다. 단순히 문법을 외우는 단계에서 벗어나 데이터를 읽고 정리하며 의미를 찾아내는 감각을 키웠습니다.

    • Java
    • Python
    • Data Analysis

  4. 2023

    첫 개발 프로젝트로 아이디어를 구현하다

    머릿속 아이디어를 실제 화면과 기능으로 옮기며 개발 프로세스를 경험했습니다. 요구사항을 작게 쪼개고, 구현하고, 다시 다듬는 과정을 통해 결과물은 생각보다 많은 선택의 합이라는 것을 배웠습니다.

    Project
    개인 웹페이지 구조 설계와 인터랙션 구현
    Stack
    HTML, CSS, JavaScript, Git
    Lesson
    작동하는 기능과 읽히는 경험은 함께 설계되어야 한다.
    • Git
    • Web
    • UI Flow

  5. 2024

    팀 안에서 사용자 경험을 중심에 두다

    팀 프로젝트를 거치며 좋은 개발은 혼자 완성하는 기술이 아니라 함께 문제를 정의하고 조율하는 과정이라는 것을 배웠습니다. 사용자가 어떤 흐름으로 서비스를 만나는지 관찰하며 UI/UX와 협업 역량을 함께 키웠습니다.

    • UI/UX
    • Collaboration
    • Problem Solving

  6. 2025

    AI와 데이터로 실제 문제에 접근하다

    AI와 데이터 기반 프로젝트를 통해 문제를 해석하고, 필요한 정보를 선별하고, 서비스 형태로 연결하는 경험을 쌓았습니다. 기술의 목적은 새로움 자체가 아니라 사람에게 닿는 가치라는 기준이 더 분명해졌습니다.

    Project
    데이터 기반 서비스 기획과 AI 활용 실험
    Stack
    AI, Python, Data Analysis, Service Planning
    Lesson
    문제의 맥락을 이해할수록 기술 선택도 선명해진다.
    • AI
    • Python
    • Service Design

  7. Now

    정보와 기술을 사람에게 연결하는 중

    현재는 Android와 Kotlin을 포함한 소프트웨어 개발 역량을 넓히며, 데이터와 사용자 경험을 함께 고려하는 개발자로 성장하고 있습니다. 작은 기능을 만들더라도 사용자의 맥락과 흐름을 먼저 생각하려고 합니다.

    • Android
    • Kotlin
    • Human-centered Tech

  8. Future

    융합으로 사회적 가치를 만드는 개발자

    앞으로는 데이터, AI, 소프트웨어를 연결해 실제 사회 문제를 해결하는 서비스를 만들고 싶습니다. 기술을 잘 다루는 사람을 넘어, 문제와 사람 사이를 정확히 잇는 개발자가 되는 것이 이 여정의 다음 목적지입니다.
