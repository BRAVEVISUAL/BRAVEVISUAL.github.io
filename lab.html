JavaScript Lab

작은 기능을 관찰하고 검증하는 웹 실험 노트입니다.

JavaScript 예제를 하나의 연구 기록처럼 정리했습니다. 각 실험은 목적, 입력, 실행, 관찰 결과의 흐름으로 이어집니다.

  1. Experiment 01

    랜덤 뽑기

    배열로 분리된 입력값에서 무작위 인덱스를 선택하는 과정을 관찰합니다.

    Hypothesis
    입력값을 배열로 변환하면 임의 선택 결과를 만들 수 있다.
    Method
    String split, trim, filter, Math.random 사용
    결과가 여기에 표시됩니다.

  2. Experiment 02

    D-Day 계산기

    현재 날짜와 목표 날짜의 차이를 계산해 시간 데이터를 화면에 해석 가능한 정보로 바꿉니다.

    Hypothesis
    두 날짜의 시간 차이를 일 단위로 변환할 수 있다.
    Method
    Date 객체, millisecond 차이, Math.ceil 사용
    목표 날짜를 선택해 주세요.

  3. Experiment 03

    DOM 스타일 제어

    사용자의 클릭이 DOM 요소의 시각 상태를 어떻게 변화시키는지 관찰합니다.

    Hypothesis
    이벤트 리스너를 통해 화면 요소의 스타일을 직접 변경할 수 있다.
    Method
    click event, style property, display toggle 사용

    이 문장의 스타일을 바꿔보세요!
